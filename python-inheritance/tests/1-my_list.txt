The ``1-my_list`` module
======================

Using ``print_sorted``
-------------------

>>> MyList = __import__('1-my_list').MyList

**Basic input**
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

**Negative input**
>>> my_list = MyList()
>>> my_list.append(-1)
>>> my_list.append(-4)
>>> my_list.append(-2)
>>> my_list.append(-3)
>>> my_list.append(-5)
>>> my_list.append(-15)
>>> my_list.append(-8)
>>> print(my_list)
[-1, -4, -2, -3, -5, -15, -8]
>>> my_list.print_sorted()
[-15, -8, -5, -4, -3, -2, -1]
>>> print(my_list)
[-1, -4, -2, -3, -5, -15, -8]

**List 1 element**
>>> my_list = MyList()
>>> my_list.append(5)
>>> print(my_list)
[5]
>>> my_list.print_sorted()
[5]
>>> print(my_list)
[5]

**Duplicate element**
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(4)
>>> my_list.append(5)
>>> my_list.append(1)
>>> my_list.append(3)
>>> my_list.append(6)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 4, 5, 1, 3, 6, 5]
>>> my_list.print_sorted()
[1, 1, 3, 4, 4, 5, 5, 6]
>>> print(my_list)
[1, 4, 4, 5, 1, 3, 6, 5]

**Empty list**
>>> my_list = MyList()
>>> print(my_list)
[]
>>> my_list.print_sorted()
[]
>>> print(my_list)
[]
